# 岷县文化宣传网站项目说明

## 项目简介
本项目是一个展示岷县文化、旅游、历史等特色的网站，采用前后端分离架构，提供丰富的文化展示和用户互动功能。

## 目录结构说明
- backend/：后端代码目录，包含Flask应用和API实现
- css/：样式文件目录，存放网站样式表
- js/：JavaScript文件目录，存放前端交互脚本
- images/：图片资源目录，存放网站使用的图片
- videos/：视频资源目录，存放网站使用的视频
- uploads/：用户上传文件存储目录
- src/：源代码目录，存放核心代码文件

## 主要页面说明
- index.html：网站首页，展示核心内容
- culture.html：文化特色展示页
- tourism.html：旅游信息展示页
- history.html：历史背景介绍页
- specialties.html：特产美食展示页
- folklore.html：民俗风情展示页
- interaction.html：用户互动页面
- about.html：关于岷县介绍页
- author.html：作者信息页面

## 技术文档
- requirements.txt：项目依赖包列表
- 项目说明文档.md：详细的项目说明文档
- README.md：项目完整说明文档

## 运行环境
- Python 3.7+
- 现代浏览器（Chrome、Firefox、Safari、Edge等）
- 网络连接

## 安装说明
1. 创建虚拟环境：python -m venv venv
2. 激活虚拟环境：
   - Windows: venv\Scripts\activate
3. 安装依赖：pip install -r requirements.txt

## 运行说明
1. 进入backend目录
2. 运行python app.py
3. 访问http://localhost:8000

## 注意事项
- 请确保已安装所有依赖包
- 运行前请确保端口8000未被占用
- 上传文件大小限制为10MB
- 建议使用Chrome浏览器访问以获得最佳体验
